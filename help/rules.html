Alecto / Rules

Alecto allows you to manage rules written in JSON with a RESTful API.

Viewing the current set of rules: GET /rules/all

Viewing a specific rule: GET <Alecto URI>/rules/<rule number>

Example: The 0th rule (if any)

Creating/replacing a rule: POST /rules

Do a http POST on /rules, with the JSON rule specification in the POST body.

Deleting a rule: DELETE /rules/<rule number>

i.e. use the DELETE http method on the <Alecto URI>/rules/<rule number> resource

Delete all rules: DELETE /rules/all

i.e. use the DELETE http method on the <Alecto URI>/rules/all resource
